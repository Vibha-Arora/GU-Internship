# JavaScript ES6+ Coding Practice Sheet

## Overview
This repository contains solutions for 25 JavaScript ES6+ practice questions.

Topics covered:
- const and let
- Block Scope
- Arrow Functions
- Array & Object Destructuring
- Template Literals
- Default Parameters
- Rest Operator
- Spread Operator
- map(), filter(), reduce()
- Mini Challenge
